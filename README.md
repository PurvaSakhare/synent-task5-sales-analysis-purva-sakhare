# Sales Data Analysis using Superstore Dataset

## Project Overview

This project analyzes the Superstore Sales Dataset to evaluate business performance and identify key trends in sales and profitability. The analysis was performed using Python in Google Colab.

## Objectives

* Analyze monthly revenue trends
* Identify top-selling products
* Perform profit analysis
* Generate actionable business insights

## Dataset

The project uses the Superstore Sales Dataset containing:

* Order Date
* Sales
* Profit
* Category
* Sub-Category
* Product Name
* Customer Information
* Region

## Analysis Performed

### 1. Monthly Revenue Trends

* Analyzed sales performance over time
* Identified revenue growth and seasonal patterns

### 2. Top-Selling Products

* Ranked products based on total sales
* Identified products contributing most to revenue

### 3. Profit Analysis

* Evaluated profit across different categories
* Compared sales performance with profitability

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Google Colab

## Project Files

* `Sales_Analysis.ipynb` – Analysis notebook
* `Sample_Superstore.csv` – Dataset
* `Business_Insights_Report.pdf` – Final report

## Key Business Insights

* Revenue varies across months, indicating seasonal trends.
* A small group of products contributes significantly to overall sales.
* High sales do not always result in high profits.
* Focusing on profitable categories can improve business performance.

## Conclusion

This analysis provides valuable insights into sales performance, product demand, and profitability. The findings can support data-driven business decisions and strategic planning.
